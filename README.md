# OZE Mazowsze

Lokalny portal o odnawialnych źródłach energii na Mazowszu.

## Stack
- Astro 4.x (static output)
- Vanilla CSS (Official Sky Blue — regional/government style theme)
- Cloudflare Pages deployment

## Uruchomienie
```bash
npm install
npm run dev
```

## Deploy
Cloudflare Pages: build `npm run build`, publish dir `dist`
